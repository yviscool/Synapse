# Synapse - Ваш AI-инструментарий

> **Управление промптами · Сбор чатов · Фрагменты кода — Всё-в-одном AI-опыт**

Synapse — это расширение для браузера, созданное для вашего AI-рабочего процесса. Оно объединяет управление промптами, сбор чатов и фрагменты кода в одном месте. Управляйте промптами с историей версий, полнотекстовым поиском и вставкой в один клик. Собирайте диалоги с 12+ платформ, включая ChatGPT, Claude, Gemini и DeepSeek, с синхронизацией в реальном времени и экспортом в различных форматах. Навигируйте по длинным диалогам с автоматически генерируемыми оглавлениями.

## ✨ Основные функции

### 📝 Управление промптами

* Создание, редактирование, категоризация и поиск промптов в одном месте
* WYSIWYG Markdown-редактор с предпросмотром в реальном времени
* Автоматическое сохранение истории версий промптов с возможностью восстановления
* Вставка промптов одним кликом рядом с полями ввода AI-сайтов (`Alt + K`)
* `Ctrl + Shift + S` для быстрого сохранения выделенного текста / контекстное меню
* Полнотекстовый поиск для быстрого доступа

### 💬 Сбор и управление чатами

* Сбор диалогов с AI-платформ одним кликом, ручная и реальновременная синхронизация
* 12+ платформ: ChatGPT, Claude, Gemini, AI Studio, DeepSeek, Kimi, Doubao, Tencent Yuanbao, Grok, Copilot, MiniMax, Zhipu ChatGLM и др.
* Фильтрация по платформе, тегам и избранному; многомерная сортировка
* Подсветка кода, формулы KaTeX и диаграммы Mermaid
* Навигация по оглавлению для быстрого перехода
* Экспорт в JSON / Markdown / TXT / HTML / PDF

### 🧩 Управление фрагментами кода

* Древовидная структура папок + теги для гибкого управления
* Подсветка синтаксиса для 30+ языков
* Избранное, поиск и многомерная сортировка
* Копирование одним кликом с автоматическим отслеживанием использования

### 🔖 Оглавление длинных диалогов

* Интеллектуальный анализ диалогов для создания структурированных оглавлений
* Быстрая навигация вверх, к центру или вниз с иконочной категоризацией
* В поле ввода на AI-сайте нажмите `Ctrl + ↑ / ↓`, чтобы быстро подставить предыдущий или следующий пользовательский запрос из структуры диалога; нажмите `Esc`, чтобы восстановить текущий черновик
* Перетаскиваемый интерфейс оглавления, обновление в реальном времени, поддержка тёмной/светлой темы

### ☁️ Данные и синхронизация

* Безопасное локальное хранение данных с поддержкой импорта/экспорта
* Облачная синхронизация через Google Drive
* Импорт и объединение данных для удобной миграции

## 🚀 Руководство по использованию

* В **поле ввода AI-сайта** введите `/p` или нажмите `Alt + K` для вызова **промпт-селектора**
* В **поле ввода AI-сайта** нажмите `Ctrl + ↑ / ↓`, чтобы быстро переключаться между предыдущими или следующими запросами из оглавления; нажмите `Esc`, чтобы восстановить текущий черновик
* Быстрое сохранение: `Ctrl + Shift + S` или выделите текст → правый клик → Сохранить как промпт
* На боковой панели AI-сайта соберите текущий диалог одним кликом или включите синхронизацию в реальном времени
* Откройте панель управления расширением для управления промптами, чатами и фрагментами кода

## 📦 Руководство по установке

### Chrome Web Store
[Установить из Chrome Web Store](https://chromewebstore.google.com/detail/synapse/mdnfmfgnnbeodhpfnkeobmhifodhhjcj?authuser=0&hl=ru)

### Microsoft Edge Add-ons
[Установить из Microsoft Edge Add-ons](https://microsoftedge.microsoft.com/addons/detail/foicpmidihacnocjmdbkjanbjncckoaa)

### Ручная установка
1. Перейдите на страницу [Releases](https://github.com/yviscool/synapse/releases)
2. Скачайте `extension-vX.X.X.zip`
3. Откройте страницу управления расширениями браузера и включите **Режим разработчика**
4. Перетащите ZIP-файл для установки
5. Нажмите на значок Synapse на панели инструментов
