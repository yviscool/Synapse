# Synapse - Tu caja de herramientas de IA

> **Gestión de Prompts · Recolección de Chats · Fragmentos de Código — Experiencia de IA todo en uno**

Synapse es una extensión de navegador para tu flujo de trabajo con IA. Integra gestión de prompts, recolección de chats y snippets de código en un solo lugar. Puedes administrar prompts con historial de versiones, búsqueda de texto completo e inserción en un clic en sitios de IA. También permite recopilar conversaciones de más de 12 plataformas como ChatGPT, Claude, Gemini y DeepSeek, con sincronización en tiempo real y exportación en múltiples formatos.

## ✨ Funciones principales

### 📝 Gestión de Prompts

* Crear, editar, categorizar y buscar prompts
* Editor Markdown WYSIWYG con vista previa en tiempo real
* Historial de versiones automático y restauración
* Inserción de prompts con un clic junto al cuadro de entrada de sitios de IA (`Alt + K`)
* `Ctrl + Shift + S` para guardar texto seleccionado rápidamente
* Búsqueda de texto completo

### 💬 Recolección y gestión de chats

* Recolección con un clic y sincronización manual o en tiempo real
* Compatible con 12+ plataformas: ChatGPT, Claude, Gemini, AI Studio, DeepSeek, Kimi, Doubao, Tencent Yuanbao, Grok, Copilot, MiniMax, Zhipu ChatGLM y más
* Filtros por plataforma, etiquetas y favoritos; ordenamiento multidimensional
* Vista detallada con resaltado de código, KaTeX y Mermaid
* Navegación por esquema para conversaciones largas
* Exportación a JSON / Markdown / TXT / HTML / PDF

### 🧩 Gestión de fragmentos de código

* Organización por carpetas y etiquetas
* Resaltado de sintaxis para 30+ lenguajes
* Favoritos, búsqueda y ordenamiento
* Copia con un clic y seguimiento de uso

### 🔖 Esquema de conversaciones largas

* Análisis inteligente y generación automática de esquema
* Navegación rápida por secciones
* En el cuadro de entrada de IA, pulsa `Ctrl + ↑ / ↓` para recuperar desde el esquema el prompt anterior o siguiente del usuario; pulsa `Esc` para restaurar tu borrador actual
* Panel arrastrable con actualización en tiempo real

### ☁️ Datos y sincronización

* Almacenamiento local con copia de seguridad por importación/exportación
* Sincronización en la nube con Google Drive
* Importación y combinación de datos para migraciones

## 🌐 Plataformas compatibles

ChatGPT · Claude · Gemini · AI Studio · DeepSeek · Kimi · Doubao · Tencent Yuanbao · Grok · Copilot · MiniMax · Zhipu ChatGLM y más

## 🚀 Guía de uso

* En el cuadro de entrada de sitios de IA, escribe `/p` o pulsa `Alt + K`
* En el cuadro de entrada de sitios de IA, pulsa `Ctrl + ↑ / ↓` para cambiar rápidamente entre prompts anteriores o siguientes del esquema, y `Esc` para restaurar el borrador actual
* Guarda rápidamente con `Ctrl + Shift + S` o clic derecho sobre texto seleccionado
* Usa el panel lateral para recolectar conversaciones o activar sincronización en tiempo real
* Abre el panel de la extensión para gestionar prompts, chats y snippets

## 📦 Instalación

### Chrome Web Store
[Instalar desde Chrome Web Store](https://chromewebstore.google.com/detail/synapse/mdnfmfgnnbeodhpfnkeobmhifodhhjcj?authuser=0&hl=es)

### Microsoft Edge Add-ons
[Instalar desde Microsoft Edge Add-ons](https://microsoftedge.microsoft.com/addons/detail/foicpmidihacnocjmdbkjanbjncckoaa)

### Instalación manual
1. Abre la página de [Releases](https://github.com/yviscool/synapse/releases)
2. Descarga `extension-vX.X.X.zip`
3. Activa **Developer mode** en la página de extensiones
4. Arrastra el archivo ZIP para instalar
5. Haz clic en el icono de Synapse para comenzar
