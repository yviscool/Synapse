# Synapse - Sua caixa de ferramentas de IA

> **Gerenciamento de Prompts · Coleta de Conversas · Snippets de Código — Experiência de IA tudo em um**

Synapse é uma extensão de navegador para o seu fluxo de trabalho com IA. Ela reúne gerenciamento de prompts, coleta de conversas e snippets de código em um só lugar. Gerencie prompts com histórico de versões, busca em texto completo e inserção com um clique em sites de IA. Colete conversas de mais de 12 plataformas, incluindo ChatGPT, Claude, Gemini e DeepSeek, com sincronização em tempo real e exportação em múltiplos formatos.

## ✨ Recursos principais

### 📝 Gerenciamento de prompts

* Criar, editar, categorizar e buscar prompts
* Editor Markdown WYSIWYG com pré-visualização em tempo real
* Histórico automático de versões com restauração
* Inserção de prompts com um clique ao lado da caixa de entrada (`Alt + K`)
* `Ctrl + Shift + S` para salvar texto selecionado rapidamente
* Busca em texto completo

### 💬 Coleta e gestão de conversas

* Coleta com um clique, manual ou em tempo real
* Suporte para 12+ plataformas
* Filtros por plataforma, tags e favoritos
* Visualização detalhada com destaque de código, KaTeX e Mermaid
* Navegação por sumário para conversas longas
* Exportação em JSON / Markdown / TXT / HTML / PDF

### 🧩 Gerenciamento de snippets de código

* Organização por pastas e tags
* Realce de sintaxe para 30+ linguagens
* Favoritos, busca e ordenação
* Cópia com um clique e rastreamento de uso

### 🔖 Sumário de conversa longa

* Análise inteligente com geração automática de estrutura
* Navegação rápida para topo, meio e fim
* No campo de entrada do site de IA, pressione `Ctrl + ↑ / ↓` para preencher rapidamente o prompt anterior ou seguinte do usuário a partir do outline; pressione `Esc` para restaurar o rascunho atual
* Painel arrastável com atualização em tempo real

### ☁️ Dados e sincronização

* Dados locais com importação/exportação para backup
* Sincronização em nuvem com Google Drive
* Importação e mesclagem de dados para migração

## 🚀 Guia de uso

* Na caixa de entrada de IA, digite `/p` ou pressione `Alt + K`
* Na caixa de entrada de IA, pressione `Ctrl + ↑ / ↓` para alternar rapidamente entre prompts anteriores ou seguintes do outline, e `Esc` para restaurar o rascunho atual
* Salve rapidamente com `Ctrl + Shift + S` ou clique com o botão direito no texto selecionado
* Use o painel lateral para coletar conversas ou habilitar sincronização em tempo real
* Abra o painel da extensão para gerenciar prompts, conversas e snippets

## 📦 Instalação

### Chrome Web Store
[Instalar pela Chrome Web Store](https://chromewebstore.google.com/detail/synapse/mdnfmfgnnbeodhpfnkeobmhifodhhjcj?authuser=0&hl=pt-BR)

### Microsoft Edge Add-ons
[Instalar pelos complementos do Microsoft Edge](https://microsoftedge.microsoft.com/addons/detail/foicpmidihacnocjmdbkjanbjncckoaa)

### Instalação manual
1. Acesse a página [Releases](https://github.com/yviscool/synapse/releases)
2. Baixe `extension-vX.X.X.zip`
3. Ative o **Developer mode**
4. Arraste e solte o arquivo ZIP para instalar
5. Clique no ícone do Synapse para começar
