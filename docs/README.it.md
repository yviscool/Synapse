# Synapse - Il tuo toolbox AI

> **Gestione Prompt · Raccolta Chat · Snippet di Codice — Esperienza AI tutto in uno**

Synapse è un'estensione browser per il tuo workflow AI. Unisce gestione prompt, raccolta chat e snippet di codice in un unico posto. Gestisci prompt con cronologia versioni, ricerca full-text e inserimento con un click sui siti AI. Raccogli conversazioni da oltre 12 piattaforme come ChatGPT, Claude, Gemini e DeepSeek, con sincronizzazione in tempo reale ed esportazione in più formati.

## ✨ Funzionalità principali

### 📝 Gestione prompt

* Crea, modifica, organizza e cerca prompt
* Editor Markdown WYSIWYG con anteprima in tempo reale
* Cronologia automatica delle versioni con ripristino
* Inserimento prompt con un click vicino al campo input AI (`Alt + K`)
* `Ctrl + Shift + S` per salvataggio rapido del testo selezionato
* Ricerca full-text

### 💬 Raccolta e gestione chat

* Raccolta con un click in modalità manuale o realtime
* Supporto a 12+ piattaforme
* Filtri per piattaforma, tag e preferiti
* Vista dettaglio con evidenziazione codice, KaTeX e Mermaid
* Navigazione a struttura per chat lunghe
* Export in JSON / Markdown / TXT / HTML / PDF

### 🧩 Gestione snippet di codice

* Organizzazione ad albero con cartelle e tag
* Evidenziazione sintassi per oltre 30 linguaggi
* Preferiti, ricerca e ordinamento multi-criterio
* Copia con un click e tracciamento utilizzo

### 🔖 Struttura per conversazioni lunghe

* Analisi intelligente e generazione automatica struttura
* Navigazione rapida in alto, centro e basso
* Nel campo di input dei siti di IA, premi `Ctrl + ↑ / ↓` per reinserire rapidamente dal sommario il prompt utente precedente o successivo; premi `Esc` per ripristinare la bozza corrente
* Pannello trascinabile con aggiornamento realtime

### ☁️ Dati e sincronizzazione

* Archiviazione locale con import/export backup
* Sincronizzazione cloud con Google Drive
* Import e merge dati per migrazione

## 🚀 Guida rapida

* Nel campo input AI, digita `/p` o premi `Alt + K`
* Nel campo input AI, premi `Ctrl + ↑ / ↓` per passare rapidamente ai prompt precedenti o successivi del sommario, e `Esc` per ripristinare la bozza corrente
* Salva velocemente con `Ctrl + Shift + S` o con menu contestuale
* Usa il pannello laterale per raccogliere chat o attivare sync realtime
* Apri la dashboard dell'estensione per gestire prompt, chat e snippet

## 📦 Installazione

### Chrome Web Store
[Installa da Chrome Web Store](https://chromewebstore.google.com/detail/synapse/mdnfmfgnnbeodhpfnkeobmhifodhhjcj?authuser=0&hl=it)

### Microsoft Edge Add-ons
[Installa da Microsoft Edge Add-ons](https://microsoftedge.microsoft.com/addons/detail/foicpmidihacnocjmdbkjanbjncckoaa)

### Installazione manuale
1. Vai alla pagina [Releases](https://github.com/yviscool/synapse/releases)
2. Scarica `extension-vX.X.X.zip`
3. Abilita il **Developer mode** nella pagina estensioni
4. Trascina il file ZIP per installare
5. Clicca l'icona Synapse per iniziare
