# Synapse - Votre boîte à outils IA

> **Gestion des Prompts · Collecte des Conversations · Extraits de Code — Expérience IA tout-en-un**

Synapse est une extension de navigateur conçue pour votre workflow IA. Elle réunit la gestion des prompts, la collecte des conversations et les snippets de code au même endroit. Gérez vos prompts avec historique des versions, recherche plein texte et insertion en un clic sur les sites IA. Collectez des conversations depuis plus de 12 plateformes (ChatGPT, Claude, Gemini, DeepSeek, etc.), avec synchronisation en temps réel et export multi-format.

## ✨ Fonctionnalités principales

### 📝 Gestion des prompts

* Créer, modifier, classer et rechercher des prompts
* Éditeur Markdown WYSIWYG avec aperçu en temps réel
* Historique de versions automatique avec restauration
* Insertion des prompts en un clic à côté du champ de saisie IA (`Alt + K`)
* `Ctrl + Shift + S` pour enregistrer rapidement du texte sélectionné
* Recherche plein texte

### 💬 Collecte et gestion des conversations

* Collecte en un clic, mode manuel ou synchronisation en temps réel
* 12+ plateformes prises en charge
* Filtrage par plateforme, tags et favoris
* Vue détaillée avec coloration du code, KaTeX et Mermaid
* Navigation par plan pour les longues conversations
* Export JSON / Markdown / TXT / HTML / PDF

### 🧩 Gestion des snippets de code

* Arborescence de dossiers + tags
* Coloration syntaxique pour 30+ langages
* Favoris, recherche et tri multi-critères
* Copie en un clic avec suivi d'usage

### 🔖 Plan de conversation longue

* Analyse intelligente et génération automatique de plan
* Navigation rapide vers le haut, le milieu ou le bas
* Dans le champ de saisie d’un site IA, appuyez sur `Ctrl + ↑ / ↓` pour réinjecter depuis le plan la requête utilisateur précédente ou suivante ; appuyez sur `Esc` pour restaurer votre brouillon actuel
* Interface glissable, mise à jour en temps réel

### ☁️ Données et synchronisation

* Stockage local sécurisé avec import/export
* Synchronisation cloud Google Drive
* Import et fusion de données pour migration

## 🚀 Guide d'utilisation

* Dans le champ IA, tapez `/p` ou appuyez sur `Alt + K`
* Dans le champ IA, appuyez sur `Ctrl + ↑ / ↓` pour basculer rapidement entre les requêtes précédentes ou suivantes du plan, puis sur `Esc` pour restaurer le brouillon actuel
* Enregistrement rapide: `Ctrl + Shift + S` ou clic droit sur texte sélectionné
* Utilisez le panneau latéral pour collecter les conversations
* Ouvrez le tableau de bord pour gérer prompts, conversations et snippets

### Installation manuelle
1. Ouvrez la page [Releases](https://github.com/yviscool/synapse/releases)
2. Téléchargez `extension-vX.X.X.zip`
3. Activez le **Developer mode**
4. Glissez-déposez le ZIP pour installer
5. Cliquez sur l'icône Synapse pour commencer
