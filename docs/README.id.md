# Synapse - Kotak alat AI Anda

> **Manajemen Prompt · Pengumpulan Chat · Snippet Kode — Pengalaman AI all-in-one**

Synapse adalah ekstensi browser untuk alur kerja AI Anda. Synapse menggabungkan manajemen prompt, pengumpulan chat, dan snippet kode dalam satu tempat. Kelola prompt dengan riwayat versi, pencarian teks penuh, dan injeksi satu klik di situs AI. Kumpulkan percakapan dari 12+ platform seperti ChatGPT, Claude, Gemini, dan DeepSeek, dengan sinkronisasi real-time dan ekspor multi-format.

## ✨ Fitur utama

### 📝 Manajemen prompt

* Buat, edit, kategorikan, dan cari prompt
* Editor Markdown WYSIWYG dengan pratinjau real-time
* Riwayat versi otomatis dan pemulihan
* Injeksi prompt satu klik di dekat kolom input AI (`Alt + K`)
* `Ctrl + Shift + S` untuk menyimpan teks terpilih dengan cepat
* Pencarian teks penuh

### 💬 Pengumpulan & manajemen chat

* Pengumpulan satu klik dengan mode manual atau sinkronisasi real-time
* Mendukung 12+ platform
* Filter berdasarkan platform, tag, dan favorit
* Tampilan detail dengan code highlight, KaTeX, dan Mermaid
* Navigasi outline untuk percakapan panjang
* Ekspor JSON / Markdown / TXT / HTML / PDF

### 🧩 Manajemen snippet kode

* Organisasi pohon folder + tag
* Highlight sintaks untuk 30+ bahasa
* Favorit, pencarian, dan pengurutan multi-dimensi
* Salin satu klik dengan pelacakan penggunaan

### 🔖 Outline percakapan panjang

* Analisis cerdas dan pembuatan outline otomatis
* Navigasi cepat ke atas, tengah, dan bawah
* Di kotak input situs AI, tekan `Ctrl + ↑ / ↓` untuk mengisi ulang prompt pengguna sebelumnya atau berikutnya dari outline; tekan `Esc` untuk memulihkan draf saat ini
* Panel dapat digeser dengan update real-time

### ☁️ Data & sinkronisasi

* Penyimpanan lokal dengan cadangan impor/ekspor
* Sinkronisasi cloud Google Drive
* Impor dan gabung data untuk migrasi

## 🚀 Panduan penggunaan

* Di kolom input situs AI, ketik `/p` atau tekan `Alt + K`
* Di kolom input situs AI, tekan `Ctrl + ↑ / ↓` untuk beralih cepat ke prompt sebelumnya atau berikutnya dari outline, dan tekan `Esc` untuk memulihkan draf saat ini
* Simpan cepat dengan `Ctrl + Shift + S` atau klik kanan pada teks terpilih
* Gunakan panel samping untuk mengumpulkan percakapan atau aktifkan sinkronisasi real-time
* Buka dashboard ekstensi untuk mengelola prompt, chat, dan snippet

## 📦 Instalasi

### Chrome Web Store
[Instal dari Chrome Web Store](https://chromewebstore.google.com/detail/synapse/mdnfmfgnnbeodhpfnkeobmhifodhhjcj?authuser=0&hl=id)

### Microsoft Edge Add-ons
[Instal dari Microsoft Edge Add-ons](https://microsoftedge.microsoft.com/addons/detail/foicpmidihacnocjmdbkjanbjncckoaa)

### Instalasi manual
1. Buka halaman [Releases](https://github.com/yviscool/synapse/releases)
2. Unduh `extension-vX.X.X.zip`
3. Aktifkan **Developer mode** di halaman ekstensi
4. Seret dan lepas file ZIP untuk menginstal
5. Klik ikon Synapse untuk mulai menggunakan
